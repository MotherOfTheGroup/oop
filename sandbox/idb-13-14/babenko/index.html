Часть кода, созданной таблицы

Тег <tr> служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки может задаваться с помощью тега <td>
Атрибут height устанавливает высоту блочных или заменяемых элементов. Высота не включает толщину границ вокруг элемента, значение отступов и полей.
Атрибут colspan устанавливает число ячеек, которые должны быть объединены по горизонтали. Этот атрибут имеет смысл для таблиц, состоящих из нескольких строк.
Цвет таблицы устанавливает атрибут bgcolor, в данном случае его значание равно 884535
