Описание тега

Тег <button> создает на веб-странице кнопки и по своему действию напоминает результат, получаемый с помощью тега <input> (с атрибутом type="button | reset | submit"). В отличие от этого тега, <button> предлагает расширенные возможности по созданию кнопок. Например, на подобной кнопке можно размещать любые элементы HTML, в том числе изображения. Используя стили можно определить вид кнопки путем изменения шрифта, цвета фона, размеров и других параметров.

Теоретически, тег <button> должен располагаться внутри формы, устанавливаемой элементом <form>. Тем не менее, браузеры не выводят сообщение об ошибке и корректно работают с тегом <button>, если он встречается самостоятельно. Однако, если необходимо результат нажатия на кнопку отправить на сервер, помещать <button> в контейнер <form> обязательно.

Пояснение иллюстрации тега

Снизу представлена демонстрация тега <button>. Эта программа автоматизирует создание простых форм на языке HTML путем предоставления пользователю графического интерфейса. При нажатии кнопки "создать элемент" с помощью Javascript генерируется новый "input" элемент с типом "text" и надписью для него. Надпись реализуется с помощью тега "p, текст для надписи предоставляется пользователем. При нажатии кнопки " Закончить и получить html код" программа оборачивает список созданных элементов в тег "form", тем самым создавая форму, и добавляыет в эту форму кнопку Отправить. Далее программа берет получившийся html код и вставляет его в специально отведённый для этого элемент "textarea". Далее пользователь имеет возможность скопировать данный код и использовать в своих целях.
