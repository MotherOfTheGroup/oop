Wiki-страница

Тег <img>.

Предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG. Адрес файла с картинкой задаётся через атрибут src.

В качестве примера была реализована возможность поворота изображения на 90 градусов.

MPA.

Multi Page Application – сокращенно MPA, в переводе на русский язык означает “Приложение на нескольких страницах”. Другими словами MPA – это web-приложение, размещенное на нескольких web-страницах, которое для обеспечения работы загружает весь необходимый код вместе с загрузкой самих страниц.
Примером MPA в данном случае служит имитация системы авторизации.
Требовалось реализовать:

  • методы взаимодействия клиент-сервер (form, ajax)
  • взаимодействие с технологиями серверной части (php, sql)
  • систему авторизации с различными правами для пользователя и администратора

Для первого входа в систему используется логин "admin" с пустым паролем. Сбросить настройки MPA до изначального уровня можно с помощью кнопки "сброс данных" на главной странице.
