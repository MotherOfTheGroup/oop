SPA - коммиты из учебного репозитория

Задание: Реализовать SPA приложение, согласно следующим требованиям:

  • Получать коммиты, находящиеся в репозитории oop/stankin с использованием API сервиса Github
  • Выполнять авторизацию пользователя в приложении через OpenAPI ВКонтакте
  • Возможность взаимодействовать с коммитами, помечая их как "избранные" (с динамическим обновлением пользоввательского интерфейса)
  • Данные о "избранных" коммитах хранятся в СУБД MySQL, выдача клиенту производится в формате JSON
  • Предварительная работа с данными на стороне клиента выполняется с использованием фреймворка jQuery и технологии AJAX
