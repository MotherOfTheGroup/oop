Тег Select

HTML тег Select используется для создания раскрывающегося списка, который появляется при щелчке мышью по элементу формы и позволяет посетителю сайта выбрать один из предопределенных вариантов. Элемент Select может содержать два и более элементов Option, определяющих доступные варианты выбора в раскрывающемся списке. Ширина списка будет определяться самым длинным текстом, указанным в теге Option.

Требования к SPA


  1. Создать имитационную форму авторизации.
    • Форма содержит логин и пароль пользователей, при вводе которых открывается доступ к товарам.
  2. SPA позволит:
      • Получить пользователю необходимую и достоверную информацию о имеющихся товарах (марка, модель, цена, кол-во товара в наличии).
      • Хранить все необходимые данные о таварах в базе JSON.
      • Предоставит возможность скрыть/показать вывод информации о товарах (при помощи Angular).
